# Ansible Multi-Server Automation Project

## Overview
This project demonstrates how to automate configuration of multiple servers using Ansible.
---

## Tools Used
- Ansible
- AWS EC2
- Linux
---

## Features
- Multi-server management
- httpd installation
- Automated deployment
---

## Workflow
1. Configure/setup custom inventory
2. Write playbook
3. Run automation
4. Deploy web server
